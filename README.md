# iLog

This was the first GUI app I ever wrote, back in 2009. I wanted to learn Objective-C. A friend of mine at Drexel worked several part-time jobs and needed a quick way to tally up the number of hours he worked in each job in a given pay period.

iLog uses the macOS calendar API to fetch the users calendars and then sums the duration of all events in a given period and displays the total per-calendar. In Preferences the user can configure the start day and durration of the period they want to track for each calendar.
